# goldjetComponents

## 如何编写组件

请参考[dumi 官网](https://d.umijs.org/zh-CN/guide)

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father`,

```bash
$ npm run build
```
